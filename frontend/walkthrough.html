Paso 2 de 4
Análisis de Salud
Cerebro procesa imágenes NDVI y térmicas para calcular un puntaje de salud 0-100 para cada campo. El análisis combina: vegetación (NDVI), estrés hídrico (térmico), suelo, y clima.
  • + Puntaje de salud 0-100 con tendencia temporal (mejorando/estable/bajando)
  • + Mapa de calor SVG — campos coloreados por salud en vista de granja
  • + Sparklines de historial en cada tarjeta de campo
  • + Detección de enfermedades por patrones NDVI anómalos
  • + Fusión multi-sensor: NDVI + térmico + suelo + microbioma
Inteligencia Ancestral con datos de sensor
Cuando un agricultor aplica una práctica ancestral (p.ej. composta de monte), Cerebro mide el impacto real en salud del campo antes y después del tratamiento. Esto es el núcleo de la propuesta FODECIJAL: validar conocimiento tradicional con tecnología.
Paso 3 de 4
Recomendaciones Regenerativas
Cerebro recomienda tratamientos 100% orgánicos y regenerativos — nunca pesticidas sintéticos. Combina inteligencia ancestral con datos de sensor para intervenciones precisas.
  • + Motor de recomendaciones basado en tipo de cultivo + estado de salud + suelo + clima
  • + 21+ métodos orgánicos documentados (10 fertilizantes, 3 tratamientos microbioma, 8 ancestrales)
  • + Calendario de Inteligencia Ancestral — alertas por temporada de Jalisco (temporal/secas)
  • + Optimizador de timing — sugiere el mejor momento para aplicar cada tratamiento
  • + Seguimiento de efectividad — mide resultado real post-tratamiento
Ejemplo: Milpa + Composta de Monte
El sistema detecta que un campo de milpa tiene NDVI bajo (0.45). Recomienda aplicar composta de monte — una práctica ancestral que mejora microbioma del suelo. Después de 30 días, el NDVI sube a 0.72. Cerebro registra: "Práctica ancestral validada con +60% mejora."
